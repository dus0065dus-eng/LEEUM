fieldset의 자식 요소
legend는 첫번째 자식요소로 웹접근성을 위해 반드시 필요

input type="text" -> 검색 부분에서 텍스트를 작성하는 부분
input type="submit" value="검색"     ->  *input type="submit" 는 검색 버튼 부분으로 value="검색"는 그 부분의
검색이라는 텍스트가 보여 검색버튼이라는 것을 알려

<select> 태그   -> 옵션 메뉴를 제공하는 컨트롤 (흔히 '드롭다운 목록' 또는 '셀렉트 박스'라고 부르는 이 컨트롤은 사용자가 원하는 옵션을 선택할 수 있는 방법)
<select> 태그의 각 옵션은 <option> 태그로 나타냄
*사용법 (value = 값)
 <option value="red">빨강</option>

*select의 옵션바 내리는 표시 없애는 속성 appearance: none;
